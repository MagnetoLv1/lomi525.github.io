#### Angular CLI?
Angular 프로젝트를 생성 및 환경설정을 구성해주는 커맨드 인터페이스 프로그램입니다.
- 프로젝트 생성
- 간단한 웹서버
- Webpack 내장

#### Angular CLI 설치하기
{% highlight bash %}
$ npm install -g @angular/cli 
{% endhighlight %}

node.js를 설치하면 자동으로 설치된다. <br>
[node.js 사이트](https://nodejs.org)
