1. PC 전자비서 페이지에 'Mobile' 스타일 추가

가. 각 전자비서 스타일에 'Mobile' 스타일 추가

 - 채팅, 알림, 목표그래프, 선물자막

나. Mobile 스타일의 최초 값은 PC의 기본 디폴트 값과 동일함

다. Mobile 스타일의 UI 및 기능은 기본적으로 PC와 동일

 - 퀵메뉴 등 모바일에서는 적용 불가능한  일부 메뉴 비노출
- Mobile 스타일에 대한 설명 추가

라. 기존 PC의 스타일을 그대로 가져올 수 있는 스타일 복사 기능 추가

2. Mobile 방송하기에 소스편집 기능 추가

가. 소스편집 기능 이란

- PC의 전자비서 소스를 모바일에도 활용 가능하도록 제공

- 기존 이미지, 텍스트 추가 기능도 소스편집 안에 포함

나. 소스편집 종류

- '소스편집 (Mobile) 메뉴구조 페이지 참고

다. 소스편집 기능은 방송 대기/방송중 모두 사용 가능

라. 소스편집 리스트에서 현재 등록/사용중인 소스 관리 가능  

2-1. Mobile 방송하기의 소스편집_전자비서  

가. PC에 저장되어 있는  Mobile 스타일값을 그대로 적용

나. Mobile에서도 일부 설정 기능 제공

다. PC와 Mobile의 값은 서로 연동 되어 반영 되어야 함

라. Mobile의 전자비서 설정 페이지는 수정이 원활 하도록 Webview로  작업

2-2. 소스편집의 이미지/텍스트 기능   

가. 이미지/텍스트 기능은 기존과 동일

나. 일부 UI, UX , Flow는 변경 필요

3. 방송하기 화면에 소스 적용 

가. 소스편집 소스는 모두 방송화면에 적용 가능

나. 대기화면/방송중 모두 적용/수정 가능

다. 방송화면에 적용 했을 시 화면 위에서 소스 이동 및 퀵메뉴 이용 가능

 [ 화면 위에서 소스 퀵메뉴 기능 ]

1. 소스 사이즈 변경 가능

2. 소스 위치 이동 가능 (Drag & Drop)

3. 소스 방향 이동 가능 ( Roatate)

3. 소스 내의 폰트 변경 가능

4. 소스 기능 작동 on/off (일시정지) 가능

5. 기능에 따라 내용 수동 초기화 가능

6. 화면에서 소스 제거


작업 영역

PC Web : 전자비서 페이지

Mobile : iOS, Android 방송하기

Mobile webview : (신규) iOS, Android 방송하기 전자비서 설정 페이지
