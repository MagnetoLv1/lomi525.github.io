  공지 테마 사용법


   BJ와 매니저가 채팅창에 명령어를 입력하면 위젯이 표시됩니다. 


  • #공지#공지내용            채팅 상단에 공지내용 표시
  • #공지#삭제                   공지 삭제
  • #공지#시간                   현재 시간 표시
  • #공지#방송시간            방송한 시간 표시
  • #공지#추천                   UP(추천)수 표시
  • #공지#추천#공지내용   UP(추천)수와 공지 표시
  • #공지#테마#테마이름   테마이름으로 테마 변경


 

  시계 테마 사용법


   BJ와 매니저가 채팅창에 명령어를 입력하면 위젯이 표시됩니다. 


  • #시간                            채팅 상단에 현재 시간 표시
  • #시간#방송시간            방송한 시간 표시
  • #시간#00:00:00            설정한 시간으로 타이머 표시 (예. 01:10:07)
  • #시간#0                         0초부터 스톱워치 표시 
  • #시간#정지                    현재시간, 타이머, 스톱워치 정지
  • #시간#시작                    정지된 시간, 타이머, 스톱워치 시작              
  • #시간#삭제                    채팅 상단에 시계 표시 삭제
  • 시간#테마#테마이름      테마이름으로 테마 변경
